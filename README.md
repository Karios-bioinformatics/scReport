# scReport

<p align="center"> <strong>Making bioinformatics results interactive.</strong> </p>

scReport is an interactive reporting ecosystem primarily designed for single-cell bioinformatics. It aims to transform scattered analysis outputs—including static figures, tables, and scripts—into unified, interactive, shareable, and reproducible scientific reports.

Starting from single-cell analysis, scReport will gradually expand to cover the core workflows of single-cell data analysis and ultimately provide comprehensive interactive reporting for the major analytical results throughout the workflow, offering a unified solution for result presentation, scientific collaboration, knowledge sharing, and reproducible research.

The project originated from the practical challenge of managing a large number of single-cell analysis results, together with inspiration from automated QC reports generated by upstream sequencing pipelines.

## Ecosystem

### scReportLite

---

**Version:** 0.4.0
**GitHub:** [Project Repository](https://github.com/Karios-bioinformatics/scReportLite)
**Zenodo:** [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20828069.svg)](https://doi.org/10.5281/zenodo.20828069)

---

#### Overview

scReportLite is an R package for generating interactive HTML reports from the fundamental results of single-cell RNA sequencing (scRNA-seq) analysis.

Rather than replacing existing single-cell analysis frameworks such as Seurat, or re-running the complete analysis workflow, scReportLite focuses on organizing analysis outputs—including result tables, dimensional reduction coordinates, QC metrics, marker gene results, and cell annotations—into interactive, traceable, and shareable reports.

For more details, please refer to the [scReportLite README](https://github.com/Karios-bioinformatics/scReportLite/blob/main/README.md).
